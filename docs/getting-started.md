## Prerequisites
- Node.js 18+
- Python 3.10+ (optional, for hardware scripts)

## Install
```bash
git clone https://github.com/kawaiiTaiga/project_SABA
cd project_SABA
```

## First Run
1. Connect your device.
2. Configure tokens/ports.
3. Try a simple command: “Rotate servo 30° clockwise”.
